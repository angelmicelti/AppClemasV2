🔧 Barra superior

  • Proyecto: Nombre del proyecto actual.
  • + Nuevo: Crea un lienzo en blanco.
  • ↩ Deshacer / Rehacer ↪: Deshace o rehace la última acción.
  • ☰ Archivo: Exportar (JSON, SVG, PNG, JPG), guardar/cargar local, importar y GitHub.
  • ❓ Ayuda: Muestra esta ventana.

🧱 Clema

  • Las clemas son bloques rectangulares con puntos de conexión arriba y abajo.
  • Botones + / − en cada extremo de la clema para añadir o quitar bloques.
  • Cada punto de conexión muestra un indicador azul al pasar el ratón.

🔌 Cables

  • Clic en un punto de conexión para iniciar un cable. Clic en otro punto para finalizarlo.
  • W: Añade un cable libre (arco) sobre el lienzo.
  • 0-9: Cambia el color del cable seleccionado (código IEC: negro, marrón, rojo, naranja, amarillo, verde, azul, violeta, gris, blanco).
  • Arrastrar un cable: Clic sobre el cable y arrastrar para moverlo. Si se suelta sobre la clema, se reposiciona automáticamente al lado más cercano.
  • Arrastrar extremos: Clic en los puntos de anclaje (rojos) del cable y arrastrar a un punto de conexión para fijarlo.
  • Seleccionar cable: Clic sobre él. Shift+clic para selección múltiple.
  • Delete: Elimina cable(s) seleccionado(s).
  • Clic derecho sobre un cable → confirmar eliminación.
  • Nodos adicionales: Doble clic sobre un cable para añadir un nodo. Los nodos se pueden arrastrar y sus tiradores blancos permiten deformar la curvatura. Clic derecho sobre un nodo para eliminarlo.
  • Un cable que pasa sobre la clema se muestra semitransparente en esa zona.
  • No se permite crear un cable entre dos puntos de la misma clema.

🔩 Componentes

  • En la barra lateral izquierda hay botones para añadir componentes.
  • Atajos de teclado:

RResistencia

DDiodo

  • Los componentes se pueden arrastrar libremente por el lienzo.
  • Clic derecho sobre un componente → Editar, Invertir, Duplicar, Alinear, Eliminar.
  • Shift+clic para seleccionar varios componentes.
  • A: Alinea horizontalmente los componentes seleccionados (mínimo 2).
  • Delete: Elimina componente seleccionado.

📝 Texto

  • T: Activa/desactiva el modo texto — clic en el canvas para colocar texto.
  • Botón "Añadir texto" en la barra lateral para colocar etiquetas en el lienzo.
  • Los textos se pueden arrastrar y seleccionar.

🖱️ Navegación

  • Shift+clic + arrastrar en espacio vacío: desplaza el lienzo (pan).
  • Rueda del ratón: Zoom.
  • H: Centra la vista en las clemas.
  • Redimensionar canvas: Sliders en la barra lateral para ancho y alto del lienzo.

🎨 Colores y grosor

  • Paleta de colores en la barra lateral para elegir el color del próximo cable.
  • Selector de grosor (stroke width) para el siguiente cable.
  • Se pueden cambiar el color y grosor de cables existentes seleccionándolos.
